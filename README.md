# KYC-Auth
Web app based on Flask using Face recognition. This app is based on the idea of Video KYC.
<br>
<br>
Features:
<br>
-OTP verirfication
<br>
-Face detection and matching with Aadhar card
<br>
-Instant verification
<br>
-Live video check
